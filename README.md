# Prince of war

Unity 2D project prepared for source control.

## Unity version

Open with Unity `6000.3.10f1` or a compatible Unity 6 editor.

## What to commit

Commit these source folders:

- `Assets/`
- `Packages/`
- `ProjectSettings/`
- `.gitignore`
- `.gitattributes`
- `README.md`

Generated folders such as `Library/`, `Temp/`, `Logs/`, `obj/`, IDE project files, local builds, and reference extraction data are excluded by `.gitignore`.

## Notes

The local folders `Original Prince of war/` and `ai_dataset/` are kept out of Git because they are reference/source data and are not required for Unity to open the project.
